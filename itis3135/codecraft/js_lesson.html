Common Terms Glossary

Click on the terms below to reveal their definitions using a jQuery UI Accordion.

Variable

A container for storing data values. In modern JavaScript, we declare these using let or const.

DOM (Document Object Model)

The programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

Function

A reusable block of code designed to perform a specific task, executed when "called" or "invoked".
