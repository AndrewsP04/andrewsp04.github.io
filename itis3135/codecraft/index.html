Welcome to Your CodeCraft Dashboard

CodeCraft is an interactive learning module designed specifically for students taking their first steps into front-end web development. Here, you will move beyond reading about code and actually interact with it.

How to Use This Guide

  1. Explore the Visualizers: Use the CSS and JS tabs above to access interactive playgrounds where you can see code changes happen in real-time.
  2. Test Your Knowledge: Once you feel comfortable with the core concepts, head over to the Knowledge Check to take a dynamically graded quiz.
  3. Reach Out: If you get stuck on a concept or need a deeper explanation of human-computer interaction principles, use the Ask Tutor form to request personalized help.
