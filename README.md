## 🛠 Project Modules

### 🎨 Responsive Web Design (FCC Certification)
These pages focus on core web standards, accessibility (WCAG), and responsive layout techniques without the use of JavaScript.
* **[Rune Menu Cards](itis3135/cards.html):** A 3D-flipping card interactive menu using pure CSS.
* **[Cellar Inventory](itis3135/inventory.html):** A styled data table for back-of-house logistics.
* **[Barkeep Documentation](itis3135/documentation.html):** Technical documentation for tavern staff.
* **[Merch Landing Page](itis3135/product.html):** A product landing page for "The Armored Growler" featuring video embeds and CTA forms.
* **[Patron Survey](itis3135/survey.html):** A standardized input form for customer feedback.

### ⚡ JavaScript Algorithms & Data Structures (Extra Credit)
A collection of five dynamic tools built using DOM-injection and vanilla JavaScript logic, following a strict "scaffold-only" HTML structure.
* **[Rune Converter](itis3135/converter.html):** A Markdown-to-HTML translator for woodland menus.
* **[Quill-Plucked Jukebox](itis3135/machine.html):** A functional drum machine with tavern-themed audio.
* **[Pub Polls](itis3135/voting.html):** A real-time voting system for the next house brew.
* **[Acorn Vault](itis3135/bank.html):** A ledger-style bank account management tool for tavern tabs.
* **[Forest Forecast](itis3135/weather.html):** A sky-glass weather tool for foraging conditions.
